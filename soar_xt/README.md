# SoarXt

SoarXt is an extensions library with useful functions and algorithms for use in the SOAR architecture.

## Installation

Add this line to your application's Gemfile:

		gem 'soar_xt'

And then execute:

    bundle

Or install it yourself as:

    gem install soar_xt

## Usage
	  spec.add_dependency 'soar_xt'
